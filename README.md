# Code Playground Project

## 概要

このリポジトリは、GitHub Copilot を活用しながら毎日コツコツと小さなプログラムを作り、プログラミングスキルを向上させることを目的とした学習用プロジェクトです。
簡単なゲームやミニアプリを定期的に作成し、実装・設計・AI 活用の経験を積みます。

## 目的

- GitHub Copilot を活用した開発に慣れる
- 小さなプロジェクトを継続的に作ることでプログラミング力を向上させる
- アルゴリズム・UI・ロジックの実装経験を増やす
- 継続的なアウトプットを行う

## ルール

- 小さくても良いので 1 つのアプリ・ゲームを作る
- 完成度より「作り切ること」を重視する
- Copilot を積極的に活用する
- 各プロジェクトは独立したディレクトリに配置する

## ディレクトリ構成

```
code-playground/
 ├ 001-othello/
 ├ 002-todo-app/
 ├ 003-snake-game/
 └ README.md
```

## 作る予定の例

- オセロゲーム
- Todo リスト
- 電卓
- スネークゲーム
- マインスイーパー
- 簡単な Web アプリ

## 開発スタイル

- 小さく作る
- AI(Copilot)と一緒に実装する
- 実験・試行錯誤を歓迎する
- 完成したら次のプロジェクトに進む

## ゴール

- 継続的な開発習慣を作る
- Copilot を使った効率的な開発スタイルを身につける
- 小さなプロジェクトの集合体としてポートフォリオにする
